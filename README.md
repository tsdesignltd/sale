# 角南商店

角南健夫の個人の持ち物を販売する、スマートフォン向けの中古品販売PWAです。

## 主な機能

- スマートフォンのカメラ・写真ライブラリから商品写真を登録
- 品名、カテゴリ、価格（無料／日本円）、説明、販売状況を編集
- 商品カードの自動生成とカテゴリ絞り込み
- PCは5〜6列、スマートフォンは2列のレスポンシブ商品一覧
- 20商品以上もページ分割せず一括表示
- 商品詳細からメールで購入相談
- データを端末内に保存
- ホーム画面への追加、オフライン表示に対応

## ローカルで確認

```sh
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

## 公開前に変更する箇所

`app.js` 冒頭の `CONTACT_EMAIL` を、実際の問い合わせ先メールアドレスへ変更してください。

トップ右上の似顔絵は `portrait.svg` です。正式な似顔絵画像を同じファイル名で差し替えるか、`index.html` の画像パスを変更してください。

## データについて

商品情報と写真は、登録に使用したブラウザの `localStorage` に保存されます。ブラウザや端末を変えるとデータは共有されません。本番で複数端末から更新する場合は、外部データベースの接続が必要です。
